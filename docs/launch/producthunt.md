
## First comment (longer, this is what people read)
```
Hey PH 👋

I built techtuate because I got tired of paying $9/mo to merge two PDFs or hitting watermark walls on "free" tools. The pitch: free PDF editing that runs entirely in your browser, no sign-up, no upload, no watermark.

What it does:
• Annotate (highlight, freedraw, text, shapes)
• Reorder, rotate, delete, duplicate pages
• Merge PDFs, split out page ranges
• Build a PDF from images
• Fill existing form fields (with optional flatten on save)

What it doesn't do:
• Edit existing text in PDFs (intentional non-goal)
• OCR on scanned PDFs (yet)

How it stays free forever:
The whole site is static, hosted on Cloudflare Pages for $0/mo regardless of traffic. There's no server processing your file, so there's no per-user cost. Which means no paywall ever.

Six more tools planned over the next 90 days (image compressor, QR code generator, password generator, JSON formatter, word counter, image-to-PDF). One per week.

Source: https://github.com/strata-coreai/techtuate
Why-free explainer: https://techtuate.com/why-free/
Compare vs paid: https://techtuate.com/vs/

Would love your feedback, especially edge cases and PDF files that break it. I'll be here all day.
```
