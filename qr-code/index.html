About this tool

Free, forever, no sign-up. Runs entirely in your browser - nothing is uploaded. Part of techtuate, a growing collection of little tools that just work.

This generates a static QR code - the code contains your exact text, with no redirect server in the middle. If the destination ever changes, regenerate it. That's the honest trade-off vs. dynamic QR services, and the reason there's no tracking.
